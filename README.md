# hanoi_tools
Hacking / Emulation for Tower of Hanoi Arcade Game

Note: You'll need to drop your refresh rate or this game becomes impossibly difficult.

* Z   - Button 1
* X   - Button 2
* C   - Button 3
* 5   - Coin 1
* 6   - Coin 2
* 7   - Bill
* F1  - Service
* F2  - SW2
* F3  - SW1
* F12 - Exit
